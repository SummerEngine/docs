## MCP API Reference

Summer Engine's Model Context Protocol (MCP) integration is live. Connect Cursor, Claude Code, or Windsurf to control the engine from your IDE. Add nodes, set properties, import assets, run the game, and debug, all through natural language.

## What You Get

- **23 focused tools**: Scene manipulation, debugging, project settings, asset import
- **Same operations as built-in chat**: AddNode, SetProp, SaveScene, ImportFromUrl. Identical behavior.
- **Lazy connection**: MCP server starts when your IDE starts; connects to the engine on first tool call
- **No API keys**: Engine uses local tokens. Run `summer login` once for account features.
