## Use the MCP API

Summer Engine's primary API is the **MCP (Model Context Protocol)** integration. It exposes 44 tools that let any MCP-compatible AI tool control the engine: add nodes, set properties, import assets, run the game, and debug. This is the same API used by the built-in chat, Cursor, Claude Code, Windsurf, and other IDEs.
