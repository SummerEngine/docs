On paid plans, you can pick a specific model from the model selector instead of using [Auto](/auto-mode/auto) or [Premium](/auto-mode/premium).

The list below reflects models currently available in the selector. It changes over time as new models ship.

| Model | Description |
| --- | --- |
| **GPT-5.4** | OpenAI's latest flagship model, great for daily use. |
| **Sonnet 4.6** | Anthropic's latest model, great for daily use. |
| **Opus 4.7** | Anthropic's smartest model at extra-high effort. Recommended for coding and long-horizon agentic tasks. |
| **Opus 4.7 Max** | Opus 4.7 at max effort. Only for the hardest problems; may overthink and burn tokens on simpler tasks. |
| **Gemini 3 Pro** | Google's latest flagship model, great for daily use. |
| **Gemini 3.1 Pro** | Google's latest model, great for daily use. |
| **Gemini Image** | Multimodal with image generation. |

<Info>
The Free plan only includes Auto. To pick a specific model, upgrade to a paid plan.
</Info>

## Auto and Premium

If you don't want to pick a model yourself:

- [Auto](/auto-mode/auto) balances intelligence, cost efficiency, and reliability for everyday tasks.
- [Premium](/auto-mode/premium) selects the most capable models for the most complex tasks.
