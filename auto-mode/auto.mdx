Auto allows Summer Engine to select models that balance intelligence, cost efficiency, and reliability. It is useful for everyday tasks.

Auto is the default model for new chats and the only model available on the Free plan.

## When to Use Auto

- Day-to-day game development: gameplay code, UI, scenes, small refactors.
- Asking questions about your project.
- Quick edits, prototyping, and exploration.

## When to Use Premium Instead

For the hardest problems — long agentic tasks, complex multi-file refactors, deep reasoning — switch to Premium. See [Premium](/auto-mode/premium).

## Picking a Specific Model

Paid users can also pick a specific model from the model selector instead of Auto or Premium. See [Models](/auto-mode/models) for the full list.

<Info>
Auto routing is opaque on purpose: the underlying model may change as we improve quality, cost, and reliability. If you need a specific model, pick it directly from the selector.
</Info>
